# 🎓 GradeHub – Student Result Management System

A modern and responsive web-based Student Result Management System built using HTML, CSS, and JavaScript. The application allows users to add student details, manage subjects and marks, calculate percentages and grades, and display results through an intuitive and user-friendly interface.

## Features

* Add student details
* Minimum 4 subjects required
* Add unlimited subjects dynamically
* Enter custom subject names
* Enter marks obtained and maximum marks (Out Of)
* Automatic total and percentage calculation
* Automatic grade generation
* Pass/Fail status determination
* Delete student records
* Responsive and modern glassmorphism UI
* Input validation and error handling

## Technologies Used

* HTML
* CSS
* JavaScript

## Grade Criteria

* A+ : 90% and above
* A : 80% – 89%
* B : 70% – 79%
* C : 60% – 69%
* D : 50% – 59%
* F : Below 50%

## How to Use

1. Enter the student's name.
2. Add subject names, marks obtained, and maximum marks.
3. Add additional subjects if required.
4. Click **Calculate Result**.
5. View total marks, percentage, grade, and pass/fail status.
6. Delete records when no longer needed.
